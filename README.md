# Landing Apple Leads

Landing premium orientada a captacion de leads para servicios de diseno web y desarrollo de aplicaciones.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- API Route para leads con validacion (Zod), honeypot y rate limit basico
- GA4 opcional via `NEXT_PUBLIC_GA_ID`

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
npm run start
```
